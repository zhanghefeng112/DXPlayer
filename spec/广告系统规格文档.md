## 1. 概述

### 1.1 系统简介

DXPlayerSDK 广告系统是一个为视频播放器提供广告展示能力的模块，支持在视频播放前插入片头广告（Pre-roll Ad）以及在用户暂停播放时展示暂停广告（Pause Ad）。

### 1.2 目标用户

- **SDK 集成方**：使用 DXPlayerSDK 开发视频应用的开发者
- **最终用户**：观看视频内容的普通用户和 VIP 用户

### 1.3 核心功能

1. **片头广告**：在正片播放前展示广告视频
2. **暂停广告**：在用户暂停播放时展示广告图片
3. **VIP 跳过**：VIP 用户可以跳过片头广告
4. **广告详情**：用户可以点击查看广告详情
5. **倒计时显示**：片头广告展示剩余播放时间

### 1.4 设计原则

- **非侵入性**：不影响现有播放器功能
- **可配置性**：支持灵活的广告配置
- **回调机制**：通过回调将交互事件通知外部
- **架构一致性**：遵循 Manager 模式，与 SubtitleManager、PlaybackSpeedManager 等保持一致

## 2. 功能需求

### 2.1 片头广告（Pre-roll Ad）

#### 2.1.1 基本功能

- **播放时机**：用户点击播放视频时，在正片播放前自动播放广告视频
- **播放方式**：使用独立的 FSPlayer 实例播放广告视频
- **自动切换**：广告播放完毕后自动切换到正片播放
- **全屏支持**：支持横竖屏模式，自动适配

#### 2.1.2 UI 元素

**右上角区域**：
- **倒计时标签**：显示广告剩余播放时间（如"15 秒"）
  - 字体：系统字体，白色，16pt
  - 背景：黑色半透明圆角矩形（圆角 5pt）
  - 位置：距离右上角 15pt
  - 更新频率：每秒更新一次

- **VIP 跳过按钮**：显示"VIP跳过该广告"文本
  - 字体：系统字体，白色，14pt
  - 背景：黄色渐变按钮（圆角 15pt）
  - 位置：倒计时标签下方 10pt
  - 尺寸：宽度自适应，高度 30pt
  - 显示条件：仅当 `isVIP = true` 时显示

**右下角区域**：
- **了解详情按钮**：显示"了解详情即可跳过广告"文本
  - 字体：系统字体，白色，14pt
  - 图标：右箭头图标
  - 背景：蓝色半透明圆角矩形（圆角 20pt）
  - 位置：距离右下角 15pt
  - 尺寸：宽度自适应，高度 40pt

#### 2.1.3 交互逻辑

| 操作 | 触发条件 | 行为 | 回调事件 |
|------|---------|------|---------|
| 倒计时结束 | `remainingTime <= 0` | 自动关闭广告，开始播放正片 | `onAdCompleted()` |
| 点击 VIP 跳过按钮 | `isVIP = true` | 立即关闭广告，开始播放正片 | `onAdSkipped(reason: .vip)` |
| 点击了解详情按钮 | 任何时候 | 关闭广告，开始播放正片，通知外部打开详情页 | `onAdDetailClicked(url: URL)` |
| 播放错误 | 广告视频加载失败 | 跳过广告，直接播放正片 | `onAdError(error: Error)` |

#### 2.1.4 状态流转

```
[初始状态] → [加载广告视频] → [播放广告] → [倒计时] → [广告结束] → [播放正片]
                    ↓              ↓
              [加载失败]     [用户跳过]
                    ↓              ↓
              [播放正片]      [播放正片]
```

### 2.2 暂停广告（Pause Ad）

#### 2.2.1 基本功能

- **显示时机**：用户暂停视频播放时立即显示
- **隐藏时机**：
  - 用户恢复播放视频
  - 用户点击关闭按钮
  - 用户点击广告图片后自行处理
- **图片加载**：支持三种图片源
  1. 网络 URL（异步加载）
  2. 本地 UIImage 对象（直接显示）
  3. Asset 资源名称（从 Assets.xcassets 加载）

#### 2.2.2 UI 元素

**正中央区域**：
- **广告图片容器**
  - 尺寸：宽度 300pt，高度 200pt（可配置）
  - 背景：白色
  - 圆角：10pt
  - 阴影：黑色 30% 透明度，偏移 (0, 2)，模糊半径 8pt
  - 位置：屏幕正中央

**右上角区域**（相对于广告图片容器）：
- **广告标签**："广告"文本
  - 字体：系统字体，白色，12pt
  - 背景：红色圆角矩形（圆角 3pt）
  - 位置：距离容器右上角内侧 8pt
  - 尺寸：宽度自适应，高度 20pt

- **关闭按钮**：X 图标
  - 图标：系统 "xmark" 图标，白色
  - 背景：黑色 50% 透明度圆形（直径 24pt）
  - 位置：广告标签右侧 8pt
  - 尺寸：24x24 pt

#### 2.2.3 交互逻辑

| 操作 | 触发条件 | 行为 | 回调事件 |
|------|---------|------|---------|
| 暂停视频 | 用户点击暂停按钮 | 显示暂停广告 | `onPauseAdShown()` |
| 恢复播放 | 用户点击播放按钮 | 隐藏暂停广告 | `onPauseAdHidden()` |
| 点击关闭按钮 | 任何时候 | 隐藏暂停广告 | `onPauseAdClosed()` |
| 点击广告图片 | 任何时候 | 通知外部打开详情页 | `onPauseAdClicked(url: URL?)` |
| 图片加载失败 | 网络图片加载失败 | 显示占位图或不显示 | `onPauseAdLoadError(error: Error)` |

#### 2.2.4 状态流转

```
[视频播放中] → [用户暂停] → [显示暂停广告] → [用户恢复播放] → [隐藏广告]
                                 ↓
                           [点击关闭按钮]
                                 ↓
                            [隐藏广告]
```

```swift
/// - Parameter config: 片头广告配置
```

**使用示例**：

```swift
let config = PrerollAdConfig(
    videoURL: URL(string: "https://example.com/ad_video.mp4")!,
    duration: 15.0,
    isVIP: userManager.isVIP,
    detailURL: URL(string: "https://example.com/ad_detail")
)

config.onCompleted = {
    print("广告播放完成")
}

config.onSkipped = { reason in
    switch reason {
    case .vip:
        print("VIP 用户跳过广告")
    case .detail:
        print("用户点击详情跳过")
    case .error:
        print("广告错误，自动跳过")
    }
}

config.onDetailClicked = { url in
    // 打开 WebView 或跳转详情页
    let webVC = WebViewController(url: url)
    self.present(webVC, animated: true)
}

config.onError = { error in
    print("广告加载失败: \(error.localizedDescription)")
}

playerContainer.adManager?.showPrerollAd(config: config)
```

### 6.1 片头广告状态机

#### 6.1.1 状态定义

```swift
private enum PrerollAdState {
    case idle           // 空闲状态
    case loading        // 加载广告视频
    case playing        // 正在播放广告
    case paused         // 广告已暂停（不常用）
    case skipped        // 广告已跳过
    case completed      // 广告播放完成
    case error          // 广告加载/播放错误
```

#### 6.1.2 状态转换图

```
         [idle]
           ↓
      (调用 showPrerollAd)
           ↓
       [loading] ─────→ [error] ───────┐
           ↓                           ↓
      (视频加载完成)               (播放正片)
           ↓                           ↑
       [playing] ────────────────────┐ │
           ↓                         │ │
      (倒计时结束)                   │ │
           ↓                         │ │
      [completed] ───────────────────┘ │
                                       │
      (VIP跳过/点击详情)               │
           ↓                           │
       [skipped] ─────────────────────┘
```

#### 6.1.3 状态处理

```swift
private func handlePrerollAdState(_ state: PrerollAdState) {
    switch state {
    case .idle:
        // 清理之前的广告
        cleanupPrerollAd()

    case .loading:
        // 显示 loading 指示器（可选）
        showLoadingIndicator()

    case .playing:
        // 隐藏 loading，显示广告覆盖层
        hideLoadingIndicator()
        showPrerollAdOverlay()
        startCountdownTimer()

    case .paused:
        // 暂停倒计时（不常用）
        pauseCountdownTimer()

    case .skipped:
        // 停止倒计时，通知回调
        stopCountdownTimer()
        currentPrerollConfig?.onSkipped?(skipReason)
        dismissPrerollAdAndPlayContent()

    case .completed:
        // 停止倒计时，通知回调
        stopCountdownTimer()
        currentPrerollConfig?.onCompleted?()
        dismissPrerollAdAndPlayContent()

    case .error:
        // 停止倒计时，通知回调，跳过广告
        stopCountdownTimer()
        currentPrerollConfig?.onError?(error)
        dismissPrerollAdAndPlayContent()
    }
}
```

#### 8.1.3 倒计时计时器实现

```swift
private var remainingAdTime: TimeInterval = 0

private func startCountdownTimer() {
    stopCountdownTimer()  // 先停止现有计时器

    remainingAdTime = currentPrerollConfig?.duration ?? 15.0

    adCountdownTimer = Timer.scheduledTimer(
        withTimeInterval: 1.0,
        repeats: true
    ) { [weak self] _ in
        guard let self = self else { return }

        self.remainingAdTime -= 1
        self.updateCountdownLabel()

        if self.remainingAdTime <= 0 {
            self.stopCountdownTimer()
            self.prerollAdState = .completed
            self.handlePrerollAdState(.completed)
        }
    }

    // 立即更新一次
    updateCountdownLabel()

    DXPlayerLogger.info("⏰ [广告] 倒计时已启动: \(remainingAdTime) 秒")
}

private func stopCountdownTimer() {
    adCountdownTimer?.invalidate()
    adCountdownTimer = nil
}

private func updateCountdownLabel() {
    let seconds = Int(ceil(remainingAdTime))
    prerollAdOverlay?.updateCountdown(seconds: seconds)
}
```

#### 8.3.3 图片加载

```swift
func loadImage(from source: ImageSource, detailURL: URL?, placeholder: UIImage?) {
    self.detailURL = detailURL

    switch source {
    case .url(let url):
        // 网络图片加载
        loadImageFromURL(url, placeholder: placeholder)

    case .image(let image):
        // 本地图片直接显示
        adImageView.image = image

    case .asset(let name):
        // Asset 图片加载
        if let image = UIImage(named: name) {
            adImageView.image = image
        } else {
            DXPlayerLogger.warning("⚠️ [广告] Asset 图片未找到: \(name)")
            adImageView.image = placeholder
        }
    }
}

private func loadImageFromURL(_ url: URL, placeholder: UIImage?) {
    // 显示占位图
    adImageView.image = placeholder

    // 异步下载
    let task = URLSession.shared.dataTask(with: url) { [weak self] data, response, error in
        guard let self = self else { return }

        if let error = error {
            DXPlayerLogger.error("❌ [广告] 图片加载失败: \(error.localizedDescription)")
            return
        }

        guard let data = data, let image = UIImage(data: data) else {
            DXPlayerLogger.error("❌ [广告] 图片数据无效")
            return
        }

        DispatchQueue.main.async {
            self.adImageView.image = image
            DXPlayerLogger.info("✅ [广告] 图片加载成功")
        }
    }

    task.resume()
}
```

```swift
        case .error: emoji = "❌"
        }

        log("\(emoji) [广告] \(message)", level: level)
    }
}

// 使用示例
DXPlayerLogger.ad("片头广告播放完成", level: .info)
DXPlayerLogger.ad("图片加载失败", level: .error)
```
